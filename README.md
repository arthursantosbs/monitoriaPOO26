1. Encapsulamento
2. Herança
3. Polimorfismo

## 🚀 Como executar

### Via terminal

```bash
javac src/*.java src/LIsta07/*.java
java -cp src Main
```

### Via IDE (IntelliJ/Eclipse/VS Code)

1. Abra a pasta do projeto.
2. Execute `src/Main.java`.
3. Navegue pelas aulas conforme o índice (`INDICE.md`).

## ✅ Objetivo deste material

Ao final, você deve conseguir:

- modelar classes com boa proteção de dados;
- reutilizar código com herança sem exageros;
- usar polimorfismo para reduzir acoplamento;
- aplicar abstração com classes abstratas e interfaces.
