PRINCIPAIS CONCEITOS:
─────────────────────────────────────────────────────

Modificadores de Acesso:
  private     → Apenas nesta classe
  protected   → Classe + subclasses
  public      → Visível em qualquer lugar
  (nenhum)    → Package-private

Palavras-Chave:
  extends         → Herança
  implements      → Implementação de interface
  abstract        → Classe/método abstratos
  super           → Referência à classe pai
  @Override       → Marca sobrescrita
  this            → Referência ao objeto atual

Estruturas de Dados:
  class           → Classe (define objeto)
  abstract class  → Classe base abstrata
  interface       → Contrato (sem implementação)
  extends         → Herança de classe
  implements      → Implementação de interface
